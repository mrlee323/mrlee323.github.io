## AI로 UI를 생성하는 시대

Vercel v0, GitHub Copilot, Claude API 등 LLM 기반 도구들이 UI 컴포넌트 생성 영역에서 놀라운 성과를 보이고 있습니다. 텍스트 설명만으로 완성도 높은 React 컴포넌트를 생성할 수 있게 되었습니다.

## Vercel v0: 텍스트에서 UI로

Vercel v0는 현재 가장 강력한 AI UI 생성 도구입니다.

```
프롬프트:
"대시보드 카드 컴포넌트를 만들어줘.
- 제목, 값, 변화율 표시
- 변화율이 양수면 초록색, 음수면 빨간색
- Tailwind CSS 사용
- TypeScript"
```

v0가 생성한 코드:

```tsx
interface MetricCardProps {
  title: string;
  value: string;
  change: number;
  unit?: string;
}

export function MetricCard({ title, value, change, unit = '%' }: MetricCardProps) {
  const isPositive = change >= 0;

  return (
    <div className="rounded-xl border bg-card p-6 shadow-sm">
      <p className="text-sm font-medium text-muted-foreground">{title}</p>
      <div className="mt-2 flex items-baseline gap-2">
        <h3 className="text-3xl font-bold">{value}</h3>
      </div>
      <div className="mt-1 flex items-center gap-1">
        <span
          className={`text-sm font-medium ${
            isPositive ? 'text-emerald-600' : 'text-red-600'
          }`}
        >
          {isPositive ? '▲' : '▼'} {Math.abs(change)}{unit}
        </span>
        <span className="text-xs text-muted-foreground">지난 달 대비</span>
      </div>
    </div>
  );
}
```

## Claude API로 동적 UI 생성

Claude API를 직접 호출해 런타임에 UI를 생성할 수도 있습니다.

```typescript
// lib/ai-ui-generator.ts
import Anthropic from '@anthropic-ai/sdk';

const client = new Anthropic();

export async function generateComponent(description: string): Promise<string> {
  const message = await client.messages.create({
    model: 'claude-opus-4-7',
    max_tokens: 2048,
    messages: [
      {
        role: 'user',
        content: `다음 설명에 맞는 React TypeScript 컴포넌트를 생성해줘.
Tailwind CSS를 사용하고, export default로 컴포넌트를 내보내줘.
코드만 반환하고 설명은 생략해.

설명: ${description}`,
      },
    ],
  });

  const content = message.content[0];
  if (content.type !== 'text') throw new Error('Unexpected response type');

  // 코드 블록에서 코드만 추출
  const codeMatch = content.text.match(/```(?:tsx?|jsx?)?\n([\s\S]+?)\n```/);
  return codeMatch ? codeMatch[1] : content.text;
}
```

## 동적 컴포넌트 렌더링

생성된 코드를 실제로 렌더링하려면 `@babel/standalone`이나 `sucrase` 같은 런타임 트랜스파일러가 필요합니다.

```tsx
// components/DynamicComponent.tsx
'use client';

import { useState, useEffect, useRef } from 'react';
import { generateComponent } from '@/lib/ai-ui-generator';

interface DynamicComponentProps {
  description: string;
}

export function DynamicComponent({ description }: DynamicComponentProps) {
  const [code, setCode] = useState<string>('');
  const [loading, setLoading] = useState(true);
  const iframeRef = useRef<HTMLIFrameElement>(null);

  useEffect(() => {
    generateComponent(description)
      .then((generatedCode) => {
        setCode(generatedCode);
        setLoading(false);
      })
      .catch(console.error);
  }, [description]);

  // iframe sandbox를 활용한 안전한 렌더링
  const iframeContent = `
    <!DOCTYPE html>
    <html>
      <head>
        <script src="https://cdn.tailwindcss.com"></script>
        <script src="https://unpkg.com/react@19/umd/react.development.js"></script>
        <script src="https://unpkg.com/react-dom@19/umd/react-dom.development.js"></script>
        <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
      </head>
      <body>
        <div id="root"></div>
        <script type="text/babel">
          ${code}
          ReactDOM.render(<App />, document.getElementById('root'));
        </script>
      </body>
    </html>
  `;

  if (loading) return <div>AI가 컴포넌트를 생성 중입니다...</div>;

  return (
    <iframe
      ref={iframeRef}
      srcDoc={iframeContent}
      sandbox="allow-scripts"
      className="w-full h-96 border rounded-lg"
      title="AI Generated Component"
    />
  );
}
```

## Streaming으로 실시간 미리보기

Claude API의 스트리밍을 활용하면 생성되는 코드를 실시간으로 확인할 수 있습니다.

```typescript
// app/api/generate-ui/route.ts
import Anthropic from '@anthropic-ai/sdk';

const client = new Anthropic();

export async function POST(request: Request) {
  const { description } = await request.json();

  const stream = await client.messages.stream({
    model: 'claude-opus-4-7',
    max_tokens: 2048,
    messages: [
      {
        role: 'user',
        content: `${description}에 맞는 React 컴포넌트를 TypeScript와 Tailwind CSS로 만들어줘.`,
      },
    ],
  });

  // ReadableStream으로 변환해서 클라이언트에 전달
  const readableStream = new ReadableStream({
    async start(controller) {
      for await (const chunk of stream) {
        if (
          chunk.type === 'content_block_delta' &&
          chunk.delta.type === 'text_delta'
        ) {
          controller.enqueue(new TextEncoder().encode(chunk.delta.text));
        }
      }
      controller.close();
    },
  });

  return new Response(readableStream, {
    headers: { 'Content-Type': 'text/plain; charset=utf-8' },
  });
}
```

## AI UI 생성의 현실적 한계

### 복잡한 상호작용

단순한 UI는 잘 생성하지만, 복잡한 드래그앤드롭, 복잡한 애니메이션은 아직 어렵습니다.

### 코드 일관성

여러 번 생성하면 매번 다른 구조와 네이밍이 나옵니다. 기존 프로젝트의 컨벤션을 강제하기 어렵습니다.

### 디자인 시스템 통합

프로젝트의 디자인 토큰, 커스텀 컴포넌트를 자동으로 활용하지 못합니다.

## 효과적인 활용 전략

1. **프로토타이핑에 활용**: 빠른 아이디어 검증에 탁월
2. **반복 코드 스캐폴딩**: 폼, 테이블, 카드 등 패턴 코드 생성
3. **인간 검토 필수**: 생성된 코드는 반드시 검토하고 프로젝트 컨벤션에 맞게 수정
4. **점진적 수정**: 한 번에 완성을 바라지 말고, 생성 → 수정 → 재생성 반복

AI UI 생성은 프론트엔드 개발자의 생산성을 크게 높일 수 있지만, 최종 결과물의 품질 책임은 여전히 개발자에게 있습니다.
