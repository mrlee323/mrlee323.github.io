## Type-safe API нҒҙлқјмқҙм–ёнҠёмқҳ мӨ‘мҡ”м„ұ

## API нҒҙлқјмқҙм–ёнҠё м„Өкі„

### кё°ліё кө¬мЎ°

```typescript
// lib/api-client.ts
import { z } from 'zod';

type ApiConfig = {
  baseUrl: string;
  headers?: Record<string, string>;
};

type RequestOptions<T extends z.ZodTypeAny> = {
  schema: T;
  method?: 'GET' | 'POST' | 'PUT' | 'PATCH' | 'DELETE';
  body?: unknown;
};

class ApiClient {
  private config: ApiConfig;

  constructor(config: ApiConfig) {
    this.config = config;
  }

  async request<T extends z.ZodTypeAny>(
    endpoint: string,
    options: RequestOptions<T>
  ): Promise<z.infer<T>> {
    const { schema, method = 'GET', body } = options;

    const response = await fetch(`${this.config.baseUrl}${endpoint}`, {
      method,
      headers: {
        'Content-Type': 'application/json',
        ...this.config.headers,
      },
      body: body ? JSON.stringify(body) : undefined,
    });

    if (!response.ok) {
      throw new ApiError(response.status, await response.text());
    }

    const data = await response.json();
    return schema.parse(data);
  }

  // нҺёмқҳ л©”м„ңл“ң
  get<T extends z.ZodTypeAny>(endpoint: string, schema: T) {
    return this.request(endpoint, { schema });
  }

  post<T extends z.ZodTypeAny>(endpoint: string, body: unknown, schema: T) {
    return this.request(endpoint, { schema, method: 'POST', body });
  }
}

class ApiError extends Error {
  constructor(public status: number, message: string) {
    super(message);
    this.name = 'ApiError';
  }
}
```

### API м„ңл№„мҠӨ л Ҳмқҙм–ҙ

```typescript
// services/user-service.ts
const UserSchema = z.object({
  id: z.string(),
  name: z.string(),
  email: z.string().email(),
});

const PaginatedUsersSchema = z.object({
  users: z.array(UserSchema),
  total: z.number(),
  page: z.number(),
  pageSize: z.number(),
});

export type User = z.infer<typeof UserSchema>;

const client = new ApiClient({ baseUrl: '/api' });

export const userService = {
  getAll: (page = 1) =>
    client.get(`/users?page=${page}`, PaginatedUsersSchema),

  getById: (id: string) =>
    client.get(`/users/${id}`, UserSchema),

  create: (data: Omit<User, 'id'>) =>
    client.post('/users', data, UserSchema),
};
```

## tRPC: лҒқм—җм„ң лҒқк№Ңм§Җ нғҖмһ… м•Ҳм „м„ұ

tRPCлҠ” л°ұм—”л“ңмҷҖ н”„лЎ нҠём—”л“ң мӮ¬мқҙмқҳ нғҖмһ…мқ„ мһҗлҸҷмңјлЎң кіөмң н•©лӢҲлӢӨ.

```typescript
// server/router.ts
import { z } from 'zod';
import { router, publicProcedure } from './trpc';

export const userRouter = router({
  getAll: publicProcedure
    .input(z.object({ page: z.number().default(1) }))
    .query(async ({ input }) => {
      return getUsers(input.page);
    }),

  getById: publicProcedure
    .input(z.string())
    .query(async ({ input: id }) => {
      const user = await getUserById(id);
      if (!user) throw new TRPCError({ code: 'NOT_FOUND' });
      return user;
    }),

  update: publicProcedure
    .input(z.object({
      id: z.string(),
      data: z.object({
        name: z.string().optional(),
        email: z.string().email().optional(),
      }),
    }))
    .mutation(async ({ input }) => {
      return updateUser(input.id, input.data);
    }),
});
```
